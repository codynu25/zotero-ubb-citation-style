# UBB Cluj - Custom Zotero CSL Style

A custom CSL (Citation Style Language) profile for Zotero, engineered to meet the official academic citation and drafting guidelines required for Bachelor's and Master's theses at Babeș-Bolyai University (UBB).

## Why this project?
Currently, there is no official template available on the Zotero Style Repository or within the open-source community that accurately reflects UBB's specific formatting requirements. This project fills that automation gap.

## Features
- Full support for Romanian academic citation standards (including quotes, italics, and specific punctuation).
- Automated handling of repeated citations (`Ibidem`, `op. cit.`) in footnotes.
- Precise mapping of metadata variables (authors, translators, volume sets, digital archives).

## How to Install in Zotero
1. Download the `babes-bolyai-university.csl` file from this repository.
2. Open Zotero and go to **Edit** > **Preferences** (on Windows) or **Zotero** > **Preferences** (on Mac).
3. Select the **Cite** tab and then click on the **Styles** sub-tab.
4. Click the **"+" (Add)** button under the list of styles.
5. Locate and select the downloaded `.csl` file. Click **Open** and confirm the installation.
